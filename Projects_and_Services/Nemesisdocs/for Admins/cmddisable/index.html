Command disabling



list all possible toggleable commands

Using /listcmds, you will receive a list with all toggleable commands.

The following commands are toggleable:

  • id
  • info
  • notes
  • afk
  • setbio
  • warns
  • locktypes
  • runs
  • bio
  • sed
  • saved
  • me
  • adminlist
  • warnlist
  • setme
  • warnfilters
  • slap
  • blacklist
  • kickme
  • filters

enabling and disabling a command

Use /enable <cmd name> or /disable <cmd name> to enable or disable it. Replace 'cmd name' with the name from /listcmds!
