RSS Feeds



Nemesis allows you to subscribe to RSS feeds in groups. Don't know what RSS is? Follow This link.

Testing a RSS feed

Using /rss <link> will print the link's data and the last entry, for testing purposes.

Adding and Removing feeds

Use /addrss <link> or /removerss <link>.

List all subscriptions

This is possible with the /listrss command.
