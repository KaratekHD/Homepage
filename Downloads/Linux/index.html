    Download Packages for GNU/Linux based Operating Systems

    You can install most of our software with the package manager of your distribution.

    Ubuntu

    Package manager installs are only supported on the x64 architecture. This guide describes how to use a package manager to install various Karatek Software (and some other cool stuff) on Ubuntu.

    Warning

    Never ever use a repository for another version of your distribution! If you don't know what version of Ubuntu you are using, fire up a terminal and run lsb_release -d

    Ubuntu 19.10 "Eoan Ermine"

    Installing our repositorys on Ubuntu 19.10 is really easy. First of all, open a terminal and make sure that you are logged in as root:

    sudo -s

    Next step is importing our key into APT:

    wget -q -O - http://download.karatek.net/repo/KEY.gpg | apt-key add -

    Then you can add the repositorys to APT:

    For Open Source Software (OSS):

    echo "deb https://download.karatek.net/repo/ubuntu/oss/eoan/ /" > /etc/apt/sources.list.d/karatek-oss.list

    For propritary software (non-free):

    echo "deb https://download.karatek.net/repo/ubuntu/non-free/eoan/ /" > /etc/apt/sources.list.d/karatek-non-free.list

    After that, update your package list:

    apt update

    Example

    As a small example, we are gonna give you instructions about how to install Teamspeak3, contained in the non-free repository: sudo apt-get install teamspeak3
